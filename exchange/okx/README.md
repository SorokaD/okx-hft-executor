# OKX Demo MVP Client

Этот пакет содержит минимальную рабочую интеграцию с OKX v5 REST для demo-торговли baseline-стратегии.

## Что реализовано

- подпись запросов (`auth.py`);
- размещение market order;
- отмена ордера;
- получение статуса ордера;
- получение открытых ордеров;
- получение позиций;
- получение account snapshot;
- получение текущей цены (`/market/ticker`);
- получение `tickSz` инструмента.

## Ограничения MVP

- только REST polling (без WebSocket);
- ориентировано на один инструмент (`OKX_INST_ID`, по умолчанию `BTC-USDT-SWAP`);
- ордера только market в baseline flow.
